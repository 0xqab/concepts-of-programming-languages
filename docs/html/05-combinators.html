Operator Method call Description
A | B A.OrElse(B) Try parsing with A.
If parsing fails, try B.
A ^ B A.AndThen(B) Parse with A.
Parse remaining input with B.
A+ A.OnceOrMore() Repeat parsing with A.
A* A.Repeated() Repeat parsing with A, zero or more times.
A? A.Optional() Parse with p or succeed
without parsing anything.
"F" Expect('F') Succeed iff the input
starts with this character.
